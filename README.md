# unSCRAWL

## Getting Started

1. Clone this repository

    ```
    $ git clone https://github.com/vasusrivastava43/unscrawl.git
    ```

1. Inside Backend folder

    ```
    $ pip install -r requirements.txt
    $ flask run
    ```

1. Insider Frontend folder

    ```
    $ flutter pub get
    $ flutter run
    ```
